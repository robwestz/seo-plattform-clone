## 🚀 Quick Start
### 1. Clone Repository
```bash
git clone https://github.com/YOUR-USERNAME/seo-intelligence-platform.git
cd seo-intelligence-platform
```
### 2. Read the Master Prompt
Open `seo-intelligence-platform/prompts/seo-platform-leader.md` and paste it into an LLM instance.
### 3. Generate Team Prompts
The Project Leader will generate 10 detailed prompts for each team.
### 4. Start Orchestration
```bash
cd seo-intelligence-platform
./start-seo-platform.sh
```
### 5. Begin Building
Open 10 LLM instances (Alpha → Kappa), paste each team's prompt, and signal:
```
BEGIN MEGA-FILE CREATION FOR SEO PLATFORM
```
